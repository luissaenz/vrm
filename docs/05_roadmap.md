# Roadmap de Desarrollo: VRM App (Cámara Atómica)

## Fase 1: MVP - "La Captura Perfecta" (Productividad)
*Objetivo: Validar el flujo hands-free y la micro-fragmentación.*

- [x] Implementación y Configuración del SDK de Flutter
- [x] Configuración de DeepSeek API para Fragmentación
- [x] Setup Inicial del Proyecto Flutter
- [ ] Implementación de la Capa de Cámara Nativa (CameraX / AVFoundation)
- [ ] Implementación del Teleprompter "Glance and Grab"
- [ ] Integración de Reconocimiento de Voz Local (KWS)
- [ ] Engine de Stitching Local (AVFoundation/MediaCodec).

## Fase 2: v1.0 - "Maestría Comunicacional" (Mejora)
*Objetivo: Integrar el coach invisible y el entrenamiento.*

- [ ] Panel de Análisis Post-Grabación (PPM, Pausas, Muletillas).
- [ ] Semáforo de Calidad y Validación Semántica.
- [ ] Módulo de Calentamiento (Entonación/Articulación).
- [ ] Seguimiento de Progreso Histórico y Nivelación Humana.

## Fase 3: Pro - "Potencia IA" (Escalabilidad)
*Objetivo: Automatización total y efectos avanzados.*

- [ ] Micro-edición automática (Zooms, Transiciones).
- [ ] Integración de B-Roll semántico mediante plugins.
- [ ] Gestión de "Inventario de Clips" (Huchas de Contenido).
- [ ] Modo Fábrica (Batching) e IA Adaptativa (Detección de fatiga).

## Fase 4: Ecosistema - "La Plataforma" (Plugins)
*Objetivo: Apertura a terceros y monetización.*

- [ ] Publicación del SDK de Plugins (SDD / Contratos OpenAPI).
- [ ] Marketplace de Plugins con Revenue Share.
- [ ] Integración con herramientas externas (Zapier, Make, CapCut).
- [ ] IA Generativa Avanzada (Lip-sync, Avatares Wan 2.2).

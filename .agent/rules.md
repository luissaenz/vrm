# Reglas del Proyecto

- **Idioma**: Todas las respuestas del asistente deben estar siempre en español, incluyendo explicaciones técnicas, documentación, comentarios en código y cualquier comunicación con el usuario.
- No se realizará ningún desarrollo a menos que haya una orden específica. Si se pide investigar, solo se entregará el resultado de la investigación. Solo se solucionarán problemas si se indica explícitamente "investiga y soluciona".
- Antes de desarrollar una sección o actualizar parte de ella, verifica los componentes existentes y, en lo posible, utilízalos en la implementación.
